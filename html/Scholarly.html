1. Abstract

Congratz is a web application, created in the light of reviving the "ritual" of making and sending cards.

This web application allows it's users to turn their creativity into something special, it allows the creation of 2 types of cards : buisness and festive, thus the application has been created with a friendly interface which can be used by mostly any users.

Besides these points, the web application was created by a group of college students written above (click here), as a project dedicated for the "Web Tehnologies" subject.

2. Introduction

This card creation application comes with these functionalities :

  • The festive cards will be able to contain diverse effects, via CSSs or GIFs;
  • These effects must be clearly delimited by a given area;
  • Examples of such effects are (visible and animated) : Snow behind a window, Flames of a candle within a halloween pumpkin.
  • These festive cards will have the ability to be sent between users, or to be sent via URLs.
  • The buisness cards will be created via a XML document, which will have the ability to be exported into a PDF file, thus making it printable.
  • The XML document will contain the name and contact data of multiple people.
  • Aside of the PDF version, the cards will be able to be exported as a vcard or a QR code, thus allowing the posibility of these being emailed.

3. The application's functions

This category is dedicated for describing the application's structure.

3.1 Homepage

The homepage is dedicated to welcoming the users, it having a minimalist, yet intriging design (HTML).

It containts 2 components : a toolbar and a animation.

The toolbar was created via CSS(view), having a gradient color changing animation (same CSS) and 2 buttons. These buttons are the menu button, which on click drops down a menu bar(CSS and JS) and a login button, which on click redirects you to the login page (JS).

The animation was created also via CSS (view), having a image changing slide show, inspiring the user to create something.

3.2 Loading Screen

The loading screen is another component of this website, it being a simple redirect page containing a css animation (HTML and CSS).

3.3 Login

The login mechanism is based of 2 modifiable text boxes, these having certain constraints (HTML and CSS).

3.4 Register

The register functionality is very similer with the login mechanism however having some small differences (HTML and CSS).
